## Projet et tâche projet
Un **N° projet** peut être indiqué en entête d’un document de vente ou d’achat.
Il est alors proposé sur les lignes.

S’il est modifié en entête, il peut être reporté sur les lignes après confirmation.

Si un projet n’a qu’une seule tâche, celle-ci est implicitement proposée lors de la saisie du « N° projet » sur une ligne de document, une ligne de feuille de saisie, ou une feuille de temps.

Les imputations préalablement définies sur les 2 axes principaux sont prises en compte pour filtrer la liste des tâches.

Sur les lignes du devis et de la commande de vente, le **N° projet** et **N° tâche projet** sont modifiables.
